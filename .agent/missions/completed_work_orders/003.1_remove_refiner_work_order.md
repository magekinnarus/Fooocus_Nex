# Work Order: Remove SDXL Refiner Logic

**Priority:** Medium  
**Estimated Effort:** 45-60 min  
**Assignee:** External agent (Gemini Flash or similar)

---

## Objective

Remove all SDXL Refiner model loading, unloading, and swap logic from `Fooocus_Nex`. Fooocus Nex does not support refiner models. The refiner code is dead code inherited from the original Fooocus.

## Scope

Only remove refiner-specific logic. Do NOT touch:
- `SDXLRefiner` class in `ldm_patched/modules/supported_models.py` (used for CLIP-G-only detection)
- `SDXLRefiner` class in `ldm_patched/modules/model_base.py` (ComfyUI core, don't modify)
- Any file under `ldm_patched/` (those are ComfyUI core modules)

## Files to Modify

### 1. `modules/default_pipeline.py` (~30 references)

**Variables to remove:**
- `final_refiner_unet` (global)
- `final_refiner_vae` (global)
- `model_refiner` (if exists)

**Functions to simplify:**
- `refresh_everything()` or equivalent: Remove `refiner_model_name` parameter and all refiner model loading logic (`refresh_refiner_model`, `synthesize_refiner_model`, etc.)
- `get_candidate_vae()`: Remove `refiner_swap_method` parameter. Remove all branches that check `final_refiner_vae`/`final_refiner_unet`. Just return `(final_vae, None)`.
- `process_diffusion()`: Remove `refiner_swap_method` parameter. Remove the `joint`/`separate`/`vae` swap logic blocks. Keep only the simple single-model sampling path (the `joint` path without refiner, which is just normal ksampler).

### 2. `modules/sample_hijack.py` (~20 references)

- The `refiner_switch()` inner function inside `sample_hacked()`
- The refiner condition preparation block (`positive_refiner`, `negative_refiner`)
- The step callback check `if step == refiner_switch_step`

**Keep:**

### 4. `modules/config.py` (few references)

### 5. `modules/flags.py` (1 reference)

### 6. `modules/core.py`

**Simplify:**

### 7. `webui.py` (if any UI elements)

- Remove any refiner model dropdown or refiner-related debug tools
- Check for refiner_swap_method UI elements and remove
